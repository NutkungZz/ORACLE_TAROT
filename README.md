# ORACLE_TAROT

A Celtic Cross Tarot reading web application. Select 10 cards and receive a formatted reading that you can copy and paste to Claude for interpretation.

## Features

- **78 Tarot Cards**: Complete deck with Major and Minor Arcana
- **Celtic Cross Spread**: Traditional 10-card layout
- **Oracle Shuffle**: Cards are shuffled by the oracle
- **Random Orientations**: Cards can appear upright or reversed
- **Copy to Claude**: Export your reading for AI interpretation

## How to Use

1. Open `index.html` in your web browser
2. Click "Begin Your Reading"
3. Select 10 cards from the shuffled deck
4. View your Celtic Cross layout
5. Copy the formatted reading text
6. Paste to Claude for your personalized interpretation

## Files

- `index.html` - Main application page
- `styles.css` - Styling and Celtic Cross layout
- `app.js` - Card selection and reading logic
- `tarot-data.js` - Complete tarot card database
- `CLAUDE.MD` - Development documentation
